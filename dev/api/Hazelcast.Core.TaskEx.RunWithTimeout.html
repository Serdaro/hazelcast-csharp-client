    RunWithTimeout Method

    RunWithTimeout(Func<CancellationToken, Task>, TimeSpan, CancellationToken)

    Runs a task with a timeout.

    Declaration
    public static Task RunWithTimeout(Func<CancellationToken, Task> taskAction, TimeSpan timeout, CancellationToken cancellationToken = default(CancellationToken))
    Parameters
    Func<CancellationToken, Task> taskAction

    The task action.
    TimeSpan timeout

    A timeout.
    CancellationToken cancellationToken

    An optional cancellation token.
    Returns
    Task

    A task that will complete when the task action completes, or the timeout expires.
    Remarks

    In case of a timeout, the TaskTimeoutException that is thrown contains a reference to the original task. Also, a continuation is added to the task, which observes its exceptions, thus making sure no unobserved exception can leak.

    Exceptions
    TaskTimeoutException

    The timeout has expired prior to the completion of the task.
